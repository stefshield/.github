# StefShield

An open-source initiative building software that prioritizes reliability, performance, and security.

Focuses on creating tools that can be trusted in real-world systems.

> Built as a foundation for production-grade open-source tools.

## Projects

Core projects in the StefShield ecosystem:

| Project | Description | Status |
|---|---|---|
| [nif-validator](https://github.com/stefshield/nif-validator) | A Java library for validating Portuguese NIF numbers | Active |

## Principles

Every StefShield project is built with:

- **Reliability** — a baseline, not a feature
- **Performance** — efficient by design and scalable in real-world use
- **Security** — considered from the start, not added later
